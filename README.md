# 🎮 猜数字游戏

一个用 C++ 写的终端互动小游戏，玩家需要在 1-100 范围内猜出程序随机生成的数字，程序会实时给出「太大/太小」的提示，直到猜对为止。

---

## 📌 项目简介
- 语言：C++
- 类型：终端小游戏
- 难度：入门级
- 运行方式：纯控制台运行（无需图形库）

---

## 🎯 游戏规则
1.  程序会在 **1 ~ 100** 之间随机生成一个数字作为答案
2.  你输入一个数字，程序会给出反馈：
    - 输入非数字字符：提示「请输入有效的数字！」
    - 数字不在 1-100 范围内：提示「数字必须在 1-100 之间！」
    - 数字比答案小：提示「太小啦！试试更大的数字~」
    - 数字比答案大：提示「太大啦！试试更小的数字~」
    - 数字等于答案：提示「🎉 恭喜你猜对了！」，游戏结束
