1.项目简介
 这是一个基于 C++ 开发的终端互动小游戏，玩家需要在指定范围内猜出程序随机生成的数字，程序会实时给出 “太大 / 太小” 的反馈，直到玩家猜对为止。
 程序随机生成 1-100 之间的目标数字
 数字不在 1-100 范围内：提示输入无效
 输入非数字字符：提示请输入有效数字
 数字小于目标值：提示 “太小啦！试试更大的数字～”
 数字大于目标值：提示 “太大啦！试试更小的数字～”
 猜对数字：提示 “🎉 恭喜你猜对了！” 并结束游戏
2.运行方法
 确保已安装 MinGW（g++ 编译器）
 打开命令行，进入项目目录：
 cd C:\Users\ASUS\Desktop\game
 g++ main.cpp -o guess_game.exe
